The contenders

  • AngularJS
  • React
  • Riot
  • Ember

AngularJS

Pros

  • Full MVC framework
  • The power of "dependency injection" (unit test)
  • A lot of magic
  • Big community
  • A lot of exciting projects!Material Angular

Cons

  • Does not play well with legacy (jQuery plugins)
  • Not compatible with IE8 since v1.3
  • Too much magic!

The future

Angular 2.O will change everything!

React

Pros

  • Modular approach, everything is a component
  • Focus on performance, fast in the browser
  • IE8 compatible

Cons

  • JSX syntax, templates are difficult to read
  • Only the View layer

The future

The project is growing fast

Riot

Pros

  • Modular approach with "custom tags"
  • Easy to learn
  • Templates easy to write and easy to code
  • A super light library (fas to download)
  • IE8 compatible

Cons

  • Very young project (Riot 2.O was released in 2015 January), some bugs, small community
  • Only the View layer
